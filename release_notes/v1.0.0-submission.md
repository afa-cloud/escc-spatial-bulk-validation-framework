# v1.0.0-submission

Initial public code-availability release for manuscript submission.

This release contains:

- public-data analysis scripts,
- configuration and requirements files,
- source-data manifests,
- processed supporting tables used for the submitted manuscript,
- reproducibility and package-audit metadata,
- citation and Zenodo metadata.

Scope limits:

- no controlled-access data,
- no full-text PDFs,
- no raw third-party datasets beyond submitted processed supporting tables,
- no manuscript drafting history or internal review notes,
- no causal, prospective clinical, or expression-drug response modeling claims.
